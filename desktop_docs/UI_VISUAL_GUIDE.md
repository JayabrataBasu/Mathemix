## Color Coding & Visual Feedback

### Diagnostic Plots Color Scheme

- **Scatter points**: Blue/Orange
- **Reference lines**: Red dashed
- **Cook's distance**: Color gradient (yellow → orange → red)
- **Normal curve**: Red solid line
- **Trend lines**: Red solid line

### Exploratory Plots Color Scheme

- **Histograms**: Blue bars
- **KDE curves**: Red line
- **Box plots**: Blue boxes, whiskers, and points
- **Violin plots**: Blue fill with black outline
- **Heatmap**: Blue (negative) → White (zero) → Red (positive)

### Button Visual States

```
Active button:     [ Button Text ]  ← Black text, normal appearance
Disabled button:   [ Button Text ]  ← Gray text, faded appearance
Hover button:      [ Button Text ]  ← Highlighted border
Pressed button:    [ Button Text ]  ← Indented appearance
```

## Tips for Visual Navigation

### 🎯 Quick Actions

1. **Double-click** variable in list → Adds to dependent field
2. **Single-click** in console → Place cursor
3. **Enter** in console → Execute command
4. **Ctrl+Click** in variable list → Multi-select

### 👁️ Visual Indicators

- **✓** in tab = Active tab
- **☑** in list = Selected variable
- **☐** in list = Unselected variable
- **•** in plot = Data point
- **─** in plot = Reference line
- **█** in heatmap = Color intensity

### 🎨 Professional Appearance

All plots use:
- Clean, modern matplotlib style
- Seaborn themes for consistency
- High-quality fonts
- Professional color schemes
- Proper spacing and alignment
